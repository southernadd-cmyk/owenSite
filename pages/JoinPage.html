Junior Web Programmer

Specification

As our Junior Web Developer you will be responsible for:
- Developing websites using HTML and JavaScript code
- Identifying & fixing small and medium sized bugs
- Adding features to existing code in our web application
- Working as part of the team on development projects

Required Qualifications and Experience:
- University degree in computer science, IT, equivalent experience or demonstrable interest in coding
- 1-3 years' experience with HTML (at least intermediate level) and MySQL databases (at least intermediate level)
- Knowledge of core web technologies, including HTML, CSS and JavaScript.
- Ability to write, read and interpret existing HTML code to understand the function and be able to debug the code
- Fluent in English

Senior Web Programmer

Specification

As our Senior Web Developer you will:
- Exemplify strong ownership and accountability
- Be a pragmatic problem-solver able to balance speed and technical debt
- Demonstrate excellent collaboration and communication with Product & Design teams
- Be curious and proactive about new technologies; eager to mentor others
- Have a strategic mindset focused on scalability, maintainability, and performance

Required Qualifications and Experience:
- 5+ years of experience building scalable websites
- Strong mastery of JavaScript / TypeScript, React, and Node.js
- Solid understanding of REST and GraphQL APIs, webhooks, and authentication
- Experience with CI/CD principles, Git, and testing practices (unit/integration)
- Experience working in Agile environments (Scrum/Kanban)
- Fluent in English
